Pour faire le CTF :
Contexte : 
Vous êtes sur un PC isolé, vous pouvez lancer ce que vous voulez dessus MAIS vous n'avez pas accès a internet, autrement dit : NE PAS UTILISER LES OUTILS.
Ici sera le challenge sera de progresser sur les outils natifs de Windows tel que PowerShell.

Etape 1 :
	- Copier-Coller le CTF sur votre bureau
Etape 2 : 
	- Ouvrir l'explorateur de fichier et rendez-vous à l'emplacement ou vous avez collé le CTF
Etape 3 :
	- Clic droit > propriété > (tout en bas) cocher la case "débloquer" de la catégorie "sécurité" (si une GPO vous bloque l'exécution)
Etape 4 :
	- Lancer le CTF
Have Fun !
